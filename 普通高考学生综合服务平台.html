数学专题练习

已知函数 f(x) = x³ - 3x + 2，求函数 f(x) 在区间 [-2, 2] 上的最大值和最小值。
最大值 4，最小值 -2
最大值 6，最小值 -2
最大值 4，最小值 0
最大值 6，最小值 0

题目解析：

首先求导数 f'(x) = 3x² - 3。令 f'(x) = 0，解得 x = ±1。计算 f(-2) = -2, f(-1) = 4, f(1) = 0, f(2) = 4。比较可得最大值为 4，最小值为 -2。正确答案为 A。
